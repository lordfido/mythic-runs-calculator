Mythic Runs Calculator

Instructions

Mythic Runs Calculator (MRC) is a small piece of software that takes your M+ group and calculates the best way to combine your characters to complete all your angular keystones.

For doing so, you need to upload a JSON file with a particular structure. You have a sample file here.

The software makes sure of having a Tank and a Healer for each stone (if available in your group), and balance the number of instances each character is doing.

If you want to know more, please visit our github.
